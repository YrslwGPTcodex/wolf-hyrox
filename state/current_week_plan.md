### GREEN version — full plan

- Mobility 5 min (squat patterns, hip + shoulder opener)

- 1 controlled set of 10 wall balls (shoulder feel + chalk practice)

#### Main fragment — 4 rounds
Each round:
- 400 m run @ 2:00 (4:45–5:00 / km), controlled, not lazy, not sprinting
- Walking Lunges 20 m @ 28 + 28 kg DB
- BBJ 20 m @ **controlled rhythm (R1 1:20, NOT 1:00 — apply BBJ limiter fix)**
- Wall Balls 25 reps @ 9 kg, target 58–65 sec
- Rest: 90 sec / 90 sec / 2:00 (R1–R3 → 90 sec each; after R3 → 2:00; after R4 → transition)

#### Final entry — 1 km run
- Pace 4:50–5:00 / km
- HR target before final WB: 145–160 bpm
- If HR > 162 and breathing chaotic: walk 20–30 sec, restart with breathing controlled
- If HR < 140: transition too long; start immediately

#### Transition discipline (mandatory — W12 diagnosis applied)
- 60–90 sec walk after 1 km
- **5–8 sec pause before first WB rep**
- Chalk hands
- Two controlled breaths
- Then start

#### Ceiling test — WB AMRAP CLEAN
- Floor: **50 reps** (re-touches W09–W11 SAT unbroken capacity)
- Realistic: **50–60** (matches W12 demonstrated fragment capacity)
- Stretch: **60+** (Brisbane peak re-touched)
- Stop rule: **TECHNICAL FAILURE**, not arbitrary number
  - Throw height drops
  - Catch becomes messy
  - Squat depth changes
  - Breathing turns panic
  - 1 ugly rep = stop
- Log clean count; optional broken add-on after rest (separate logging)
