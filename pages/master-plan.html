Weekly template:

  • MON: Reset walk or light speed economy
  • TUE: Lower + Sled (gym format, working loads)
  • WED: Upper + Z2 bridge
  • THU: HYROX Rhythm 2 rounds (controlled)
  • FRI: Aerobic Reset (long Z2)
  • SAT: WB Development 4×25 + finisher
  • SUN: Aerobic recovery or walk

Race Day Structure (HYROX Solo Pro)

8 km running interleaved with 8 stations:

  1. Run 1 km → SkiErg 1000 m
  2. Run 1 km → Sled Push 50 m
  3. Run 1 km → Sled Pull 50 m
  4. Run 1 km → Burpee Broad Jumps 80 m
  5. Run 1 km → Row Erg 1000 m
  6. Run 1 km → Farmers Carry 200 m
  7. Run 1 km → Sandbag Lunges 100 m
  8. Run 1 km → Wall Balls 100 reps

For 1:15:00:

  • 8 km running: ~32–34 min @ 4:00–4:15 / km
  • Station time total: ~38–42 min
  • Transitions ×8: ≤15 sec each
  • WB final: 5:00–5:30
